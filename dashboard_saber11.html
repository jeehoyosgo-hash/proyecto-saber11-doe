Fórmula aplicada para el cálculo de n

Muestreo aleatorio simple con corrección por población finita
n = ⌈ (N · Z² · S²) / (N · e² + Z² · S²) ⌉ = ⌈ (7,109,704 · 1.96² · 1968.5) / (7,109,704 · 9 + 1.96² · 1968.5) ⌉ = 841

Como N = 7,109,704 >> n = 841, el factor de corrección finita (1 + n/N) ≈ 1, por lo que ambos valores coinciden.

Verificación

Supuestos del Modelo

Evaluación de los tres supuestos del ANOVA: normalidad de residuales, homocedasticidad e independencia. El modelo es válido con las aclaraciones indicadas.

◎

Normalidad de residuales

Prueba de Lilliefors (KS corregida, adecuada para n > 50): D = 0.0524, p = 0.031. Se rechaza normalidad al 5%.

Sin embargo, con n = 327, el Teorema Central del Límite garantiza robustez del ANOVA. La violación es leve (D pequeño). Como alternativa no paramétrica existe Kruskal-Wallis.

⚠ Leve violación — ANOVA robusto por TCL
◈

Homocedasticidad

Prueba de Levene (center = median) sobre los 6 grupos (naturaleza × jornada):

F(5, 321) = 1.511, p = 0.186. No se rechaza igualdad de varianzas al 5%. Las varianzas son homogéneas entre grupos.

✓ Supuesto cumplido — p = 0.186
⬡

Independencia de residuales

Los datos provienen de una muestra aleatoria estratificada, no de una serie temporal. El gráfico de secuencia de residuales no muestra patrones sistemáticos.

Se asume independencia de los errores, lo cual es razonable dado el proceso de muestreo aleatorio.

1

Diseño muestral estadísticamente justificado

Se aplicó muestreo estratificado por estrato socioeconómico con n = 841 estudiantes, calculado a partir de una muestra piloto (S² = 1968.5) y un margen de error de ±3 puntos al 95% de confianza. El estimador de razón resultó más eficiente que el estimador simple (CV 0.78% vs 0.88%).

2

Puntaje global promedio estimado: 250.7 puntos

La estimación con el paquete survey dio una media de 250.74 (IC 95%: [246.42, 255.06]), dentro del margen de error definido. La distribución por estrato evidencia la desigualdad estructural del sistema educativo colombiano: mayor estrato → mayor puntaje.

3

Diferencia significativa entre colegios oficiales y no oficiales

El DBCA detectó F = 53.7 (p = 1.88×10⁻¹²) para el factor naturaleza del colegio. Los colegios no oficiales superan en promedio 41.1 puntos a los oficiales, diferencia confirmada por Tukey HSD con IC [-52.13, -30.07] que no contiene el cero.

4

El bloqueo por jornada estaba justificado

La jornada resultó significativa (F = 16.1, p = 2.16×10⁻⁷), confirmando que su inclusión como bloque mejoró la precisión del análisis al aislar esa fuente de variación del error residual. Sin el bloque, el CME habría sido mayor y la prueba F menos potente.

5

Limitaciones del estudio

El diseño es observacional — no se puede establecer causalidad. Hay desbalance en jornada NOCHE (7 obs. en no oficial). La muestra se obtuvo por offset secuencial de la API, lo que podría introducir sesgos si los registros tienen orden sistemático. No se controló por período de presentación.
